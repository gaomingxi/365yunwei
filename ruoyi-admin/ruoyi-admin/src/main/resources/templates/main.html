后台管理框架

一直想做一款后台管理系统，看了很多优秀的开源项目。于是利用空闲休息时间走上了一条不归路，基于优秀开源项目开始定制一套自己后台系统；再次感谢开源项目[抱拳][抱拳]

  1. 核心框架：Spring Boot。
  2. 安全框架：Apache Shiro。
  3. 模板引擎：Thymeleaf。
  4. 持久层框架：MyBatis。
  5. 定时任务:Quartz。
  6. 数据库连接池：Druid。
  7. 工具类：Fastjson。
